üöÄ Getting Started

VFIED API provides culturally-aware food recommendations that adapt to location, mood, dietary preferences, and cultural context.

Base URL

https://api.vfied.com

Quick Test

curl -X POST https://api.vfied.com/v1/recommend \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "location": {"city": "London", "country": "UK", "country_code": "GB"},
    "mood_text": "tired but celebrating",
    "dietary": ["vegetarian"]
  }'

üîë Authentication

All API requests require an API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Get your API key from the dashboard.

POST /v1/menus

Upload your restaurant menu to get personalized recommendations from your own items.

Request Body

{
  "menu": [
    {
      "name": "Ugali Mayai",
      "emoji": "üç≥",
      "country": "Kenya",
      "country_code": "KE",
      "category": "comfort",
      "tags": ["vegetarian", "protein-rich", "traditional"],
      "macros": {
        "kcal": 420,
        "protein_g": 22,
        "carbs_g": 60,
        "fat_g": 10
      },
      "price": "8.99",
      "currency": "GBP",
      "availability": "in_stock",
      "menu_link": "https://restaurant.com/menu/ugali-mayai"
    }
  ],
  "mode": "snapshot"
}

Response

{
  "success": true,
  "summary": {
    "accepted": 1,
    "rejected": 0,
    "total": 1
  },
  "menu_version": "v_1629123456789",
  "processing_time_ms": 234
}

Mode Options:

  • snapshot - Replace entire menu (default)
  • patch - Update only provided items

Use Case 1: Food Delivery App

// User opens app in London, feeling tired

  method: 'POST',
  headers: { 'Authorization': 'Bearer ' + apiKey },
  body: JSON.stringify({
    location: { city: 'London', country: 'UK', country_code: 'GB' },
    mood_text: 'tired after work',
    dietary: ['vegetarian'],
    budget: 'medium'
  })
});

// Returns: "Vegetable Curry" with cultural context

Use Case 2: Restaurant Menu Personalization

// First, upload your menu
await fetch('/v1/menus', {
  method: 'POST',

});

// Then get personalized recommendations

  method: 'POST',
  headers: { 'Authorization': 'Bearer ' + apiKey },

  })
});

// Returns recommendation from YOUR menu items
